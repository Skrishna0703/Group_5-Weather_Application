Weather Dashboard

The Weather Dashboard project focuses on creating an interactive and user-friendly application that allows users to fetch and display weather data for a city of their choice. By integrating a public weather API such as the OpenWeatherMap API, the application retrieves real-time weather details like temperature, humidity, and current weather conditions based on the city name entered by the user. The dashboard will feature an input field for users to type the city name and a button to initiate the search. Upon submission, the application will fetch the relevant data and dynamically display it on the webpage.

Additionally, the dashboard will maintain a list of recent searches, providing users quick access to weather information for previously searched cities. This recent search list can include clickable entries, enabling users to fetch updated weather details with ease. For a polished user experience, the dashboard will implement responsive styling using CSS, ensuring usability across various devices. The project will also incorporate robust error handling to manage cases such as invalid city names, network issues, or API errors gracefully. Advanced features like temperature toggles (Celsius/Fahrenheit) and weather icons can further enhance the functionality and visual appeal of the dashboard. Overall, the Weather Dashboard combines real-time data integration with intuitive design, making it a practical and engaging project.
